# KRDS Breadcrumb(경로 표시) 가이드라인

> 출처: KRDS (한국형 웹 콘텐츠 접근성 지침)
>
> HANUI Breadcrumb 컴포넌트는 이 가이드라인을 준수합니다.

## 브레드크럼이란?

브레드크럼(Breadcrumb)은 사용자의 현재 위치를 계층 구조로 보여주는 네비게이션 컴포넌트입니다.

## 용례

### 사용하기 적합한 경우

- 깊은 계층 구조의 웹사이트
- 전자상거래 카테고리 탐색
- 문서 사이트

### 사용하기 적합하지 않은 경우

- 단일 레벨 사이트
- 2단계 이하의 간단한 구조

## 사용성 가이드라인

### 1. 홈부터 시작

첫 번째 항목은 항상 홈(또는 최상위)이어야 합니다.

### 2. 현재 페이지는 링크 없음

마지막 항목(현재 페이지)은 링크가 아닌 텍스트로 표시합니다.

### 3. 적절한 구분자 사용

슬래시(/) 또는 화살표(>) 등 명확한 구분자를 사용합니다.

## 접근성 체크리스트

- [ ] nav 요소로 마크업했는가?
- [ ] aria-label="breadcrumb"를 제공하는가?
- [ ] 순서가 있는 목록(ol)을 사용했는가?
- [ ] 현재 페이지에 aria-current="page"가 있는가?
