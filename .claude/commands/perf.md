프로젝트의 성능 이슈를 점검해줘.

## 점검 항목

1. **렌더링:**
   - 불필요한 리렌더링
   - 레이아웃 스래싱 유발 속성 사용 여부
   - will-change / transform 남용

2. **메모리:**
   - 이벤트 리스너 정리 누락
   - 타이머/인터벌 정리 누락
   - 큰 객체 참조 유지

3. **네트워크/에셋:**
   - 이미지/에셋 최적화 (크기, 포맷)
   - 불필요한 의존성
   - 번들 크기

4. **일반:**
   - resize/scroll 이벤트 debounce/throttle 여부
   - CSS 애니메이션 vs JS 애니메이션 적절성

## Output

- 문제별 심각도 (높음/중간/낮음) 표시
- 구체적인 수정 코드 제안
