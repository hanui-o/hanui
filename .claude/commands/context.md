# Load Project Context

HANUI 디자인 시스템 프로젝트의 전체 컨텍스트를 로드합니다.

## 수행할 작업

1. **프로젝트 개요 파악**
   - README 및 프로젝트 구조 확인
   - 주요 기술 스택 및 아키텍처 이해

2. **개발 문서 읽기**
   - `/apps/docs/src/app/docs/` 디렉토리의 모든 문서 확인
   - 컴포넌트 가이드라인 및 사용법 숙지

3. **활성 GitHub 이슈 확인**
   - 현재 열려있는 이슈 목록 조회
   - 진행 중인 작업 및 우선순위 파악

4. **최근 커밋 히스토리**
   - 최근 변경사항 및 개발 방향성 이해

5. **주요 컴포넌트 구조**
   - `/packages/react/src/components/` 의 주요 컴포넌트 파악
   - 디자인 시스템 패턴 및 컨벤션 이해

## 출력 형식

프로젝트 컨텍스트 로딩 완료 후 다음 정보를 요약해서 제공:

- 📦 프로젝트 개요
- 🔥 현재 활성 이슈 (Open Issues)
- 📝 최근 주요 변경사항
- 🎯 다음 작업 추천사항
