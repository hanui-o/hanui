접근성(a11y)을 검사해줘.

## 점검 항목

1. **시맨틱 HTML:**
   - 적절한 heading 계층 (h1 → h2 → h3 순서)
   - landmark 요소 사용 (header, nav, main, section, footer)
   - 의미 없는 div/span 남용 여부

2. **ARIA:**
   - 인터랙티브 요소에 aria-label / aria-describedby
   - 동적 콘텐츠 aria-live 영역
   - role 속성 적절성

3. **키보드 네비게이션:**
   - 탭 순서 (tabindex) 적절성
   - 포커스 표시 (outline) 제거 여부
   - 키보드로 모든 인터랙션 가능한지

4. **시각:**
   - 색상 대비 비율 (WCAG AA 기준 4.5:1)
   - 이미지 alt 텍스트
   - 텍스트 크기 및 줌 대응

5. **애니메이션:**
   - prefers-reduced-motion 미디어 쿼리 대응 여부
   - 자동 재생 애니메이션 제어 가능 여부

## Output

- 항목별 pass/fail 표시
- fail 항목에 대한 구체적 수정 코드 제안
