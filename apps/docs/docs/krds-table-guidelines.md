# KRDS Table(표) 가이드라인

> 출처: KRDS (한국형 웹 콘텐츠 접근성 지침)
>
> HANUI Table 컴포넌트는 이 가이드라인을 준수합니다.

## 표란?

표(Table)는 구조화된 데이터를 행과 열로 정리하여 보여주는 컴포넌트입니다. 데이터 간의 관계를 명확히 하고 비교와 분석을 용이하게 합니다.

## 용례

### ✅ 사용하기 적합한 경우

- 구조화된 데이터 표시
- 여러 항목 비교
- 데이터베이스 레코드 나열

### ❌ 사용하기 적합하지 않은 경우

- 레이아웃 목적으로 사용하지 마세요
- 3개 이하의 간단한 항목은 리스트 사용

## 사용성 가이드라인

### 1. 명확한 헤더 제공

모든 열에 명확한 헤더를 제공하여 데이터의 의미를 전달합니다.

### 2. 정렬 기능 제공

사용자가 원하는 기준으로 데이터를 정렬할 수 있도록 합니다.

### 3. 반응형 디자인

모바일에서는 카드 형태로 전환하거나 가로 스크롤을 제공합니다.
