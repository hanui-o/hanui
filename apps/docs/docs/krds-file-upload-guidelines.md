# KRDS File Upload(파일 업로드) 가이드라인

> 출처: KRDS (한국형 웹 콘텐츠 접근성 지침)
>
> HANUI File Upload 컴포넌트는 이 가이드라인을 준수합니다.

## 파일 업로드란?

파일 업로드(File Upload)는 사용자가 로컬 파일을 시스템에 업로드할 수 있게 하는 컴포넌트입니다.

## 용례

### ✅ 사용하기 적합한 경우

- 문서 제출
- 프로필 사진 업로드
- 첨부파일 추가

### ❌ 사용하기 적합하지 않은 경우

- 텍스트 입력으로 충분한 경우
- 외부 URL로 대체 가능한 경우

## 사용성 가이드라인

### 1. 지원 파일 형식 명시

허용되는 파일 형식과 크기 제한을 명확히 표시합니다.

### 2. 드래그 앤 드롭 지원

클릭 외에도 드래그 앤 드롭을 지원하면 편리합니다.

### 3. 업로드 진행 상황 표시

파일이 업로드되는 동안 진행 상황을 보여줍니다.

### 4. 업로드 실패 시 명확한 에러 메시지

왜 실패했는지, 어떻게 해결할 수 있는지 안내합니다.

## 접근성 체크리스트
