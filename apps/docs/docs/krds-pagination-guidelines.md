# KRDS Pagination(페이지네이션) 가이드라인

> 출처: KRDS (한국형 웹 콘텐츠 접근성 지침)
>
> HANUI Pagination 컴포넌트는 이 가이드라인을 준수합니다.

## 페이지네이션이란?

페이지네이션(Pagination)은 많은 콘텐츠를 여러 페이지로 나누어 탐색할 수 있게 하는 네비게이션 컴포넌트입니다.

## 용례

### ✅ 사용하기 적합한 경우

- 검색 결과가 많을 때
- 게시판 목록
- 데이터 테이블

### ❌ 사용하기 적합하지 않은 경우

- 항목이 20개 이하인 경우
- 무한 스크롤이 더 적합한 소셜 피드

## 사용성 가이드라인

### 1. 현재 페이지 명시

현재 위치를 시각적으로 명확히 표시합니다.

### 2. 이전/다음 버튼 제공

숫자 외에도 이전/다음 버튼을 제공합니다.

### 3. 키보드 네비게이션

Tab과 Enter로 모든 페이지에 접근 가능해야 합니다.

---

## 접근성 체크리스트

- [ ] nav 요소로 마크업했는가?
- [ ] aria-label="페이지네이션"을 제공하는가?
- [ ] 현재 페이지에 aria-current="page"가 있는가?
- [ ] 키보드로 탐색 가능한가?
