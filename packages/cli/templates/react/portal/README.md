# HANUI Portal Template

KRDS 기반 민원 포털 템플릿입니다.

## 시작하기

```bash
# 개발 서버 실행
pnpm dev

# 프로덕션 빌드
pnpm build

# 프리뷰
pnpm preview
```

## 기술 스택

- React 18
- TypeScript
- Vite
- HANUI Components (@hanui/react)
