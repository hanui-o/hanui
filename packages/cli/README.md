# create-hanui-app

> CLI for creating HANUI projects

HANUI 기반 공공 웹사이트 프로젝트를 빠르게 생성하는 CLI 도구입니다.

## 사용법

```bash
npx create-hanui-app my-project
```

또는

```bash
npm install -g create-hanui-app
create-hanui-app my-project
```

## 템플릿

- **Portal Template** - 민원 포털 사이트 (Next.js)
- **Admin Template** - 관리자 페이지 (React + Vite)

## 기능

- KRDS 기반 컴포넌트 자동 설정
- TypeScript 설정
- Tailwind CSS + KRDS 통합
- ESLint + Prettier 설정
- Git 초기화
