# @hanui/react

> React components for Korean Government Design System (KRDS)

공공 웹사이트 개발을 위한 React 컴포넌트 라이브러리입니다.

## 설치

```bash
npm install @hanui/react
# or
pnpm add @hanui/react
# or
yarn add @hanui/react
```

## 사용법

```tsx
import { Button } from '@hanui/react';
import '@hanui/react/styles.css';

function App() {
  return (
    <Button size="medium" variant="primary">
      신청하기
    </Button>
  );
}
```

## 컴포넌트

- Button
- Input
- Card
- Table
- Modal
- Select
- Pagination
- Breadcrumb
- FileUpload

(개발 중)
